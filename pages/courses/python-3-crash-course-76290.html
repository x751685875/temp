Updated in [March 06th, 2023]

This course provides an introduction to the Python 3 programming language. Participants will learn how to execute Python files, understand the basics of math, variables, syntax, and comments, and become familiar with common data types, indexing, formatting, and files. Additionally, participants will learn about list comprehensions, loops, and functions, as well as packages, classes (OOP) and try/except. By the end of the course, participants will have a better understanding of the Python 3 language and be able to apply it to their own projects.

[Applications]
After completing this course, students should be able to apply their knowledge of Python 3 to create programs and scripts. They should be able to use math, variables, syntax, and comments to create programs. They should also be able to use common data types, indexing, formatting, and files to create programs. Additionally, they should be able to use list comprehensions, loops, and functions to create programs. Finally, they should be able to use packages, classes (OOP) and try/except to create programs.
