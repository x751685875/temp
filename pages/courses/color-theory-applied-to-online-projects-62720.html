Updated in [March 06th, 2023]

This course provides an overview of color theory and how it can be applied to online projects. Students will learn the definition of color and why we view specific hues the way we do. They will also learn color terminology to help them communicate and show their thoughts more effectively.

The course will then cover how to create a color palette by combining colors using color harmony. Students will then learn how to use a color palette to create an illustration and a website. Finally, the course will provide some pointers on how to correct incorrect color combinations and how to create and use color effectively in call-to-action buttons.

[Applications]
The application of this course can be seen in the creation of online projects. After completing the course, students will have a better understanding of colour theory and how to use it to create a colour palette. They will also be able to use colour effectively in illustrations and websites, as well as in call-to-action buttons. Additionally, they will be able to identify and correct incorrect colour combinations. This knowledge can be applied to any online project, from creating a website to designing a logo.

[Career Paths]
1. Graphic Designer: Graphic designers create visual concepts, using computer software or by hand, to communicate ideas that inspire, inform, and captivate consumers. They develop the overall layout and production design for various applications such as advertisements, brochures, magazines, and corporate reports. With the increasing demand for digital content, graphic designers are now expected to be proficient in digital design tools such as Adobe Photoshop, Illustrator, and InDesign.

2. Web Designer: Web designers create the look, layout, and features of a website. They are responsible for the visual aspect of a website, including its layout, color scheme, and font. They also create web page graphics, animations, and other visual elements. With the rise of mobile devices, web designers must also be proficient in responsive design, which allows a website to be optimized for different screen sizes.

3. UX/UI Designer: UX/UI designers are responsible for creating user-friendly interfaces for websites and applications. They must understand the user’s needs and create a design that meets those needs. They must also be proficient in user research, user testing, and usability testing. With the increasing demand for mobile applications, UX/UI designers must also be proficient in mobile design.

4. Digital Marketer: Digital marketers are responsible for creating and executing digital marketing campaigns. They must be proficient in SEO, content marketing, social media marketing, and email marketing. They must also be knowledgeable about the latest trends in digital marketing and be able to use analytics to measure the success of their campaigns.
